# Brand Color Palette

## Colors
- **#bdd1ca** - Soft Sage (Low Priority, Diagnostic, Review Exams)
- **#788c5d** - Olive (Medium Priority, Practice)
- **#d87758** - Coral (High Priority, Study Mode)
- **#c46685** - Dusty Rose (Critical, Test Mode, Priority icons)
- **#cbc9db** - Lavender Gray (Secondary accent)
- **#6a9bcc** - Soft Blue (Tertiary accent)

## Usage Notes
- Dashboard action cards: Keep clean borders, no background tints
- Priority badges: Use sage (Low), olive (Medium), coral (High), dusty rose (Critical)
- Priority icons (exclamation marks): Use dusty rose
- Exam generator cards: Use colors for borders and light backgrounds
- Review exams: Diagnostic=sage, Practice=olive for section identity
- Breathing animation: Sage color
