# Brand Color Palette

## Primary Colors (Original 4)
- **#bdd1ca** - Soft Sage (Diagnostic, Review Exams, Priority #1)
- **#788c5d** - Olive (Practice, Priority #2)
- **#d87758** - Coral (Study Mode, Priority #3)
- **#c46685** - Dusty Rose (Test Mode, Priority icons)
- **#e2dacd** - Warm Cream (Bullet points, accents)

## Extended Colors (New)
- **#cbc9db** - Lavender Gray
- **#6a9bcc** - Soft Blue

## Usage Notes
- Dashboard action cards: Keep clean borders, no background tints
- Priority badges: Use sage, olive, coral in sequence
- Priority icons (exclamation marks): Use dusty rose
- Exam generator cards: Use colors for borders and light backgrounds
- Review exams: Diagnostic=sage, Practice=olive for section identity
- Breathing animation: Sage color
