# Muscle Testing — Reading the Subconscious Yes/No

Source: Ch 2 (Sway Test introduction), Ch 5 (full muscle testing chapter). Multiple methods; choose what works for the situation.

## The signal

Whatever method, the body has a **stronger / weaker** or **forward / backward** response. Strong/forward = yes (congruent, true). Weak/backward = no (incongruent, false).

## Method 1 — Sway Test (primary, no partner needed)

**Setup**
- Quiet room. No music or TV.
- Stand with feet shoulder-width apart, hands at sides.
- Eyes closed if comfortable.
- Body will micro-oscillate — that's the postural-muscle baseline.

**Procedure**
1. Make a statement, e.g. "unconditional love".
2. Keep mind clear of other thoughts.
3. Within ~10 seconds body should sway **forward**.
4. Now state "war". Body should sway **backward**.

**Baseline calibration**
- True statement: "My name is [your real name]" → forward.
- False statement: "My name is [any other name]" → backward.

**Test for trapped emotion**
- "I have a trapped emotion." → forward = yes, backward = no.

**Cautions**
- If mind wanders to recent fight/argument while testing a positive statement, body may sway backward (responding to the memory).
- Practice required; some adults take longer than children to release conscious control.
- This is the **VR-friendly** method. Headset's positional tracking can detect the sway directly.

## Method 2 — Basic Arm Test (testing another person)

1. Subject stands, holds one arm out horizontally.
2. Tester places two fingers lightly on subject's wrist (little-finger side, just above wrist bone). Other hand can rest on opposite shoulder.
3. Tester says: "I'll have you make a statement, then press down. Resist gently." ("baby pressure" — finesse not force).
4. Subject states name. Tester smoothly increases downward pressure over ~3 seconds from 0 → ~5-10% of available strength.
5. True statement → arm stays locked.
6. False statement → arm gives way.

Tips:
- If subject pushes too hard, ask them to "turn down the dimmer".
- Skeptical or unwilling subjects test poorly — don't force it.

## Method 3 — Leg Length Test (table work)

1. Subject lies face down, shoes on.
2. Tester places thumbs under shoe arches, two fingers V-ing over each ankle.
3. Press both shoes into "standing flat" position.
4. Find a landmark on each heel (weld line works).
5. Say "yes" → press → check landmarks. Say "no" → press → landmarks shift, sometimes up to half an inch difference.

Used in chiropractic-style sessions where subject is already on a table.

## Method 4 — Elbow Test (shoulder-limited subjects)

Same as Arm Test but subject keeps elbow tucked, arm bent 90°, forearm parallel to floor. Tester presses on end of forearm.

## Method 5 — Hand Solo / Finger Test (self-testing, discreet)

1. Two adjacent fingers, same hand. Forefinger + middle, or middle + ring.
2. One = "resistance finger" (held out at ~5% strength).
3. Other = "pressure finger" (presses down on top knuckle of resistance finger).
4. Say "yes" → resistance finger stays locked.
5. Say "no" → resistance finger collapses.

Useful when you can't move much (in a chair, on a massage table, etc).

## Method 6 — Ring Test, O-Ring, etc.

Variations mentioned in passing. Same principle: subconscious strong/weak shows in any small muscle group.

## Troubleshooting — untestable

If neither true nor false statements yield clear responses, **you are untestable right now**. Common causes:

| Cause | Fix |
|---|---|
| Dehydration | Drink water, retry |
| Misaligned cervical vertebra | Use surrogate testing, or get adjustment |
| Physical impairment | Use surrogate testing |
| Skeptical mindset (subject) | Don't continue |
| Recent meal too heavy / blood sugar issue | Reschedule |
| Strong projection / wishful thinking | Step away, return neutral |

## Establishing a baseline (the discipline)

**Every session, every method.** Before any test of "is there a trapped emotion", get clear yes/no responses to known-true and known-false statements first. This is non-negotiable in the protocol.
