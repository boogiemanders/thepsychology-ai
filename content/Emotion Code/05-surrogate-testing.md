# Surrogate Testing and Proxy Testing

Source: Ch 9 "Surrogate Testing and Distance Healing by Proxy".

## When to use which

| Subject is... | Use |
|---|---|
| Present in the room, can muscle-test directly | Direct testing (default) |
| Present but untestable (infant, child, animal, unconscious, weak, dehydrated, in pain) | **Surrogate testing** |
| Not physically present (across town, across the world) | **Proxy testing** |

Both are described in the book as ethical, real, and producing identical results to direct testing.

## Surrogate Testing

A surrogate is a third party (or the practitioner themselves) within ~6 feet of the subject, whose body is muscle-tested in the subject's place.

### Distance constraint

- The subject's electromagnetic field is described as extending ~6 feet in all directions.
- Surrogate must be inside that field.
- **No physical touch required.** Within 6 ft is enough.

### Who can be a surrogate

Anyone who is testable. Parent, partner, friend, the practitioner — anyone.

### Procedure (acting as both tester and surrogate, self-test)

1. Feel love + gratitude + ask for divine assistance (book's framing; you can swap for "set intention").
2. Baseline-test yourself for clear yes/no.
3. Be within 6 ft of subject.
4. Connect: have subject say "My name is [their name]." (Or you say "Your name is [their name]" if they can't speak.)
5. Self-test. Should be strong/forward.
6. Confirm: have subject say a false name. Self-test. Should be weak/backward.
7. If clear: proceed with normal Emotion Code funnel (Steps 3-6 of [[03-protocol-release-session]]).

### Procedure (third-person surrogate)

1. Same intent-setting.
2. Baseline the **surrogate**.
3. Surrogate sits within 6 ft of subject (can hold hands; not required).
4. Subject says their name. Test surrogate. Strong.
5. Subject says false name. Test surrogate. Weak.
6. Continue Emotion Code funnel testing on the surrogate's body.

### Release on a surrogate

Sweep the magnet down the **subject's** back if possible. If subject is unreachable (in a coma, behind machinery), sweep down the **surrogate's** back instead. Both work.

### Common surrogate-testing scenarios

- **Infant**: parent acts as surrogate.
- **Child**: parent, or any adult the child trusts. Always get parent permission first.
- **Animal**: handler / owner acts as surrogate. Direct questions to the animal.
- **Unconscious / coma**: family member or practitioner as surrogate. Subconscious is still active in coma.
- **Pain / dehydration / cervical misalignment**: temporary untestability fixed by surrogate.

## Proxy Testing (Distance Healing)

Subject is **not present**. Proxy "becomes" the subject energetically.

### Procedure (self-as-proxy)

1. Intent: love, gratitude, divine assistance.
2. Get your own baseline.
3. **Establish connection**: state "My name is [subject's full name]." Test yourself.
   - First attempt usually tests weak (you're not connected yet).
   - Repeat until you test strong on the statement. That's the connection forming.
4. **Confirm connection**: state your own name. Should now test weak (you've taken on subject's energy).
5. Run normal Emotion Code funnel testing on your own body. Whatever you find IS in the subject.
6. Release on yourself (magnet down your Governing Meridian). The release transfers to the subject.
7. **Break the connection** (see below).

### Procedure (third-person proxy)

Same, but you test the third person. Have them say "My name is [subject's name]" until they test strong. Have them say their own name; should test weak. Proceed.

### Distance is irrelevant

Book asserts: works across continents. "Energy has no boundaries like physical bodies do."

### Breaking the connection (mandatory)

- After session ends, have proxy state their own name. Should test strong.
- If still weak, repeat until strong.
- Takes <10 seconds.
- **If skipped**: proxy may continue feeling subject's emotions for hours/days. The book documents one case where this happened until the connection was manually broken later.
- Beginners always break manually. Experienced practitioners' subconscious learns to break automatically.

## Permission rules

Always required before testing **or** before connecting energetically.

| Subject | Permission from |
|---|---|
| Conscious adult | Subject themselves |
| Unconscious adult | Their medical proxy |
| Minor | Parent / legal guardian |
| Pet / animal | Owner |

Skipping permission is described as both ethically wrong and energetically counterproductive.
