# Type I vs Type II error EPPP practice question

**TikTok script · EPPP**

## Draft

E triple P sample question. Read carefully.

A researcher rejects the null hypothesis. In reality, there is no effect. What type of error occurred?

A) Type I error
B) Type II error
C) Random sampling error
D) Systematic bias

Most people go straight for B. That is the trap the E triple P is counting on.

Type I error is a false positive. Think about convicting an innocent person.

You found an effect that does not exist. That is a false positive. That is Type I.

B is wrong because Type II error means failing to detect a real effect, not inventing a fake one.

The researcher found something that was not there. False positive. The answer is A.
