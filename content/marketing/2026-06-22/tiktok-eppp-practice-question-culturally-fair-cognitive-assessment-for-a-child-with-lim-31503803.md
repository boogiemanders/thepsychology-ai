# EPPP Practice Question: Culturally fair cognitive assessment for a child with limited English

**TikTok script · EPPP**

## Draft

Here's the ONE framework I use to pick the right assessment tool for any population question on E triple P.

A school psychologist evaluates a 10-year-old refugee child whose English is limited. The psychologist wants an instrument that minimizes cultural content and verbal demands and can be interpreted using either the C-H-C model or Luria's model. Which instrument fits? A: Kaufman Assessment Battery for Children, Second Edition. B: Peabody Picture Vocabulary Test Fifth Edition. C: Wonderlic Cognitive Ability Test. D: Fagan Test of Infant Intelligence.


The answer is A. The K-A-B-C two was built for cultural fairness. It minimizes verbal demands and gives you a choice of C-H-C or Luria interpretation frameworks. B is wrong because the P-P-V-T measures receptive vocabulary only, not broad cognitive processing. C is wrong because the Wonderlic is a brief group test for employment, not a child cognitive battery. D is wrong because the Fagan was designed for infants, not school-age children.
