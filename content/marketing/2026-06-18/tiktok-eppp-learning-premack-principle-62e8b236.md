# EPPP Learning: Premack Principle

**TikTok script · EPPP**

## Draft

Every parent has used this behaviorism technique without knowing its clinical name. It shows up on the E triple P every single year.

A behavioral therapist is working with an adolescent who refuses to do math homework but spends hours playing video games. The therapist recommends that parents allow video game time only after the adolescent finishes math homework. This is an application of:

A. The Premack principle, because a high-frequency behavior is used to reinforce a low-frequency behavior
B. Shaping, because the therapist is reinforcing successive approximations toward homework completion
C. Stimulus control, because the presence of homework signals that reinforcement is available
D. A token economy, because video game time functions as a token that can be exchanged for reinforcement

The answer is A, the Premack principle. High-frequency behavior is the reinforcer. Low-frequency behavior is what you want more of. Video games are high-frequency. Math homework is low-frequency. That is the reinforcer.

B is wrong because shaping requires reinforcing gradual steps toward a target behavior. This is one step, not successive approximations.
C is wrong because stimulus control is about a cue that signals when behavior will be reinforced. The homework is not a cue here, it is a requirement.
D is wrong because a token economy uses a symbolic currency. Video game time is the reward itself, not a token.
