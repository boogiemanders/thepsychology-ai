# Base Rate Neglect: EPPP Assessment Practice Question

**TikTok script · EPPP**

## Draft

Most E triple P candidates miss this question. Even people scoring ninety on practice tests.

A clinician screens patients for depression. The tool has ninety percent sensitivity and eighty percent specificity. The base rate in this clinic is ten percent. A patient tests positive. What is the actual probability they have depression?

A: ten percent.
B: ninety percent.
C: eighty percent.
D: thirty-three percent.

A is wrong because ten percent is just the pre-test base rate.

B is wrong because ninety percent is sensitivity, not what a positive result means.

C is wrong because eighty percent is specificity.

The answer is D: thirty-three percent. When prevalence is low, even an accurate test produces more false positives than true positives. This is base rate neglect, and it is on the real E triple P.
