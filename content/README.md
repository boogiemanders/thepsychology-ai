# Content Folder

Organized by project. Each project has `strategy/` (where it's going) and `research/` (evidence behind it).

## Projects

- `eppp/` — EPPP exam prep platform (live, primary revenue). iOS app docs live in `eppp/app/`.
- `insurance-matching/` — client-to-psychologist matching + insurance verification + booking (building).
- `psychotherapy/` — psychotherapy research + future AI-assisted therapy planning (early).
- `homepage/` — future homepage that ties the above projects together.

## Cross-project

- `company/` — vision, messaging rules, partnerships.
- `founder-drafts/` — rough writing, idea dumps, non-core content experiments.
- `Inzinna/` — work product for Inzinna practice (SP Notes, payroll, leadership timeline).
- `music/` — creative lab.

## Rule of thumb

- Future feature or roadmap → `<project>/strategy/`
- Market research, competitive teardown, pain points → `<project>/research/`
- Cross-project vision or messaging → `company/`
- Rough writing that shouldn't drive product decisions → `founder-drafts/`
