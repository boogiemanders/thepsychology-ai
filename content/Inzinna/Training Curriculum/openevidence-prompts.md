# OpenEvidence Prompts — IFS & EMDR Didactics

Use these prompts in OpenEvidence (https://www.openevidence.com) to build literature-based didactic content for the v1 Didactics Workstream modules. Run each prompt in a **fresh chat** to keep source citations clean. Copy DOIs/PMIDs into your slide-deck reference list — citations directly address the 6/7 accuracy-skepticism barrier surfaced in the clinician survey.

**Framing for both modules:** End each deck with a "What this training is NOT" slide. Anders is not IFS-I or EMDRIA certified; modules are positioned as literature-based orientations, not certification training.

---

## Module 1 — IFS (Internal Family Systems) Foundations

### Prompt 1 — Evidence base & empirical status
```
What is the current empirical evidence base for Internal Family Systems (IFS)
therapy as of 2026? Summarize: (1) RCTs and meta-analyses, (2) APA/SAMHSA/NICE
evidence rating, (3) populations and conditions with the strongest evidence
(PTSD, depression, chronic pain, rheumatoid arthritis), (4) known limitations
of the evidence base (sample sizes, study quality, replication). Cite primary
sources with DOIs where available.
```

### Prompt 2 — Core model & terminology
```
Explain the core theoretical model of IFS in clinical terms: the multiplicity
of mind, Self-energy, the three-part taxonomy of parts (managers, firefighters,
exiles), the concept of burdens, and unblending. Provide precise operational
definitions a licensed psychologist can use clinically. Compare to ego-state
therapy and structural dissociation models.
```

### Prompt 3 — Clinical protocol & phases
```
Walk me through the IFS clinical protocol step by step: how a trained IFS
therapist moves from contact with a protector, to permission, to unburdening
an exile. Include the "6 F's" (Find, Focus, Flesh out, Feel toward, beFriend,
Fears) and session structure. Describe what an experienced IFS session looks
like at the micro-process level.
```

### Prompt 4 — Indications, contraindications, risks
```
For what clinical presentations is IFS indicated and contraindicated? What are
the known risks or adverse events (e.g., destabilization in dissociative
disorders, psychosis, active suicidality)? What stabilization or screening
should precede IFS work? How does IFS interact with pharmacotherapy?
```

### Prompt 5 — Integration with other modalities
```
How does IFS integrate with CBT, DBT, EMDR, psychodynamic therapy, and
somatic approaches? Where do the paradigms conflict and where do they
complement? Provide examples of integrated case conceptualization.
```

### Prompt 6 — Case formulation example
```
Give an illustrative case formulation of a hypothetical adult patient with
complex PTSD, chronic self-criticism, and binge eating — written in IFS
language. Identify likely manager/firefighter/exile configurations, typical
burdens, and a 12-session treatment arc.
```

### Prompt 7 — Training pathways & scope-of-practice
```
What are the formal IFS training pathways (IFS Institute Levels 1/2/3,
IFS-I certification)? What is appropriate scope-of-practice for a clinician
who is NOT IFS-certified but wants to integrate IFS-informed concepts
(parts work, Self-energy) into generalist practice? Cite IFS Institute
position statements.
```

## Module 2 — EMDR Orientation

### Prompt 1 — Evidence base & mechanism
```
What is the current empirical evidence base for EMDR as of 2026? Summarize
(1) RCTs and meta-analyses vs. trauma-focused CBT and prolonged exposure,
(2) APA/WHO/VA-DoD guideline recommendations for PTSD, (3) proposed
mechanisms of action (working memory taxation, REM-like reconsolidation,
interhemispheric coordination) and which have empirical support, (4)
evidence outside PTSD (depression, phobias, chronic pain, addiction). Cite
primary sources.
```

### Prompt 2 — 8-phase protocol
```
Describe the EMDR 8-phase protocol in clinical detail: (1) History/treatment
planning, (2) Preparation, (3) Assessment, (4) Desensitization, (5)
Installation, (6) Body scan, (7) Closure, (8) Re-evaluation. For each phase,
specify the clinical task, expected duration, decision points, and what
constitutes phase completion. Include the standard protocol script.
```

### Prompt 3 — Assessment & target selection
```
How does an EMDR therapist identify and sequence targets? Explain the
"touchstone" concept, the Adaptive Information Processing (AIP) model,
negative/positive cognitions (NC/PC), Validity of Cognition (VOC) scale,
Subjective Units of Disturbance (SUDS), and floatback/affect scan target
identification. Include the standard NC/PC cognition lists.
```

### Prompt 4 — Bilateral stimulation parameters
```
What does the evidence show about bilateral stimulation parameters —
eye movements vs. taps vs. tones, set length, speed, number of sets?
What's required vs. optional for efficacy? Cite dismantling studies.
```

### Prompt 5 — Indications, contraindications, stabilization
```
For which clinical presentations is EMDR indicated and contraindicated?
Who requires extended Phase 2 preparation (e.g., complex PTSD, dissociative
disorders, somatic vulnerability)? Describe the resourcing techniques
(Calm/Safe Place, Container, Resource Development and Installation).
What are reported adverse events and abreaction-management procedures?
```

### Prompt 6 — Variants for complex presentations
```
Summarize major EMDR protocol variants: R-TEP (Recent Traumatic Episode),
G-TEP (Group Traumatic Episode), EMDR 2.0, Progressive Approach (Gonzalez
& Mosquera), Attachment-Focused EMDR (Parnell), and Flash Technique.
Which have empirical support? Which are indicated for complex/dissociative
populations vs. single-incident trauma?
```

### Prompt 7 — Case formulation example
```
Give an illustrative EMDR case conceptualization and target sequence plan
for a hypothetical adult patient with single-incident PTSD from a motor
vehicle accident. Include touchstone memory selection, NC/PC pairs,
SUDS/VOC targets, and a session-by-session treatment plan.
```

### Prompt 8 — Training pathways & scope-of-practice
```
What are the formal EMDR training pathways (EMDRIA-approved Basic Training,
Consultation, Certification, Consultant-in-Training)? What is appropriate
scope-of-practice for a clinician who has NOT completed EMDRIA Basic
Training but wants to understand and refer appropriately? Cite EMDRIA
position statements on unregulated practice.
```

## Deck-Building Workflow (per module)

1. Run all 7–8 prompts in fresh OpenEvidence chats
2. Dump each response into a markdown working file in this folder (`openevidence-raw-IFS.md`, `openevidence-raw-EMDR.md`)
3. Pull 2–3 slides' worth of content from each prompt response
4. Target deck length: ~20 slides for a 60-minute module (3-min/slide average with discussion)
5. Required opening slide: "Scope & framing — this is a literature-based orientation, not IFS-I / EMDRIA certification training"
6. Required closing slide: "What this training is NOT" + referral pathway to credentialed providers
7. Commit the final `.md` outline or exported slide assets into `content/Inzinna/Training Curriculum/`
