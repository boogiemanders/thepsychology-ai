### Pay Calculation
1. Look up rate per session using payer + CPT code from tables above
2. Sum all session rates for the pay period = **session total**
3. Session total x **0.80** = Bret's session pay
4. Add **supervision pay** on top (calculated from supervisee payroll — Izzy, Carlos, Karen)
5. Session pay + supervision pay = **grand total** (entered into JustWorks)
