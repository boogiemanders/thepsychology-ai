SimplePractice self-pay scraper — bookmarklet

SETUP (once):
1. Make a new bookmark (bookmark any page, then edit it).
2. Name it: "Scrape self-pay".
3. Paste the entire line below as the URL/Location. Save.

USE (each cycle, zero screenshots):
1. Go to https://secure.simplepractice.com/clients?billingType=Self-pay
2. Click the "Scrape self-pay" bookmark.
3. It auto-scrolls, then downloads self-pay-roster.json (also copies to clipboard).
4. Move that file into this folder (sosa-audit/), replacing the old roster.

Note on couples: this list page may show a couple as a short display name
(e.g. "Marissa W. & Danny W."). If that differs from the appointment report's

add the correct spelling once. Nothing is silently dropped.
